    Internet

    Internet est un réseau de réseaux d'ordinateurs où circulent des données. Les machines échangent des requêtes. Celles qui envoient les requêtes sont appelées clients et celles qui répondent serveurs. Leurs liens peuvent être filaires ou non.
      Dates clées
    • 1966 : Premier réseau internet américain (ArpaNet)
    • 1972 : Premier réseau internet français (Cyclades)
    • 1978 : Protocole TCP/IP
    • 1980 : Minitel
    • 1983 : Choix du nom d'internet
    • 1989 : World Wide Web (WWW)
    • 1997 : Wi-Fi
    • 2000 : 3G
    • 2010 : 4G
    • 2020 : 5G

    1. Donner la définition de ‘Internet’.
    Internet est un système qui connecte de nombreux réseaux informatiques locaux entre eux.



    2. Nommer 2 types de liaisons permettant la transition de données à travers le réseau Internet. Donner des exemples pour chaque type de liaison.
    Plusieurs types de liaisons permettent la circulation des données dans ce réseau :
    - les liaisons filaires telles que les câbles réseau ou la fibre optique,
    - et les liaisons hertziennes comme le Wi-Fi, la 4G ou la 5G.



    3. Nommer 2 activités de loisirs auxquels est dû en grande partie le trafic Internet.
    Une part importante du trafic Internet provient des loisirs numériques, notamment le streaming vidéo et les jeux en ligne.



    4. Quel appareil permet la communication entre 2 réseaux d’ordinateurs à travers Internet ?
    Les données sont dirigées à travers de multiples routeurs pour assurer la communication entre les réseaux.



    5. Qu’est-ce que le protocole IP et quel est son rôle ?
    Le protocole IP permet de transporter les données sur le réseau, en les faisant passer d’un appareil émetteur à un appareil récepteur. Chacun possède une adresse IP unique qui permet au protocole de les localiser.



    6. Qu’est-ce qu’une adresse IP ? Donner un exemple (visible sur la vidéo).
    Une adresse IP est une suite de chiffres difficile à mémoriser.
    Exemple : 169.153.203.240



    7. Qu’est-ce que le protocole TCP et quel est son rôle ?
    Le protocole TCP est responsable de la bonne transmission des données. Il définit les règles de découpage, d’envoi, de vérification et de réassemblage des informations entre deux ordinateurs.



    8. Pourquoi les adresses symboliques ont été créées ?
    Comme les adresses IP sont composées de chiffres compliqués à retenir, les adresses symboliques ont été inventées pour faciliter l’accès aux ressources sur Internet.



    9. Quel est le rôle d’un serveur DNS ?
    Le serveur DNS analyse une adresse symbolique pour trouver l’adresse IP correspondante en interrogeant d’autres serveurs. Il fonctionne comme un annuaire et transmet ensuite à l’utilisateur l’adresse IP du serveur demandé, permettant ainsi l’accès à la ressource.



    10. Nommer les 2 grands modèles d’architecture d’échanges de données sur Internet. Expliquer le fonctionnement de chaque modèle.
    - Modèle client-serveur : les données sont stockées sur un serveur, et les clients (utilisateurs) envoient des requêtes pour y accéder.
    - Modèle pair à pair (peer-to-peer) : chaque participant du réseau peut être à la fois client et serveur. Les machines stockent les données localement et s’échangent directement des requêtes.



    11. Quel est l’avantage et l’inconvénient du modèle pair-à-pair au modèle classique ?
    Ce modèle permet un téléchargement plus rapide et évite de surcharger un serveur unique.
    Cependant, il est souvent critiqué pour son usage dans des échanges de contenus protégés par le droit d’auteur.

    Le web

    1. Que désigne ‘Le Web’ ?
    Le web désigne un ensemble de ressources liées les unes aux autres par des liens hypertextes, c’est-à-dire tous les éléments sur lesquels on clique pour passer d'une page à une autre.



    2. Quelle est la condition essentielle pour votre ordinateur, pour que vous puissiez accéder au web ?
    Pour pouvoir y accéder, une machine doit être connectée à Internet.



    3. Où sont stockées toutes les ressources du web ?
    Toutes les ressources du web sont stockées sur des serveurs distants.



    4. Expliquer le processus des échanges client-serveur.
    Pour pouvoir y accéder et les consulter, un client doit envoyer des requêtes à un serveur. Celui-ci peut répondre en retour en fournissant des codes de pages web, des images, des vidéos, etc. Ces échanges entre les clients et les serveurs sont régis par des protocoles.



    5. Nommer des protocoles qui régissent les échanges entre le client et le serveur.
    Le protocole HTTP notamment, ou sa version sécurisée : le protocole HTTPS.



    6. Qu’est-ce qu’un URL ? De quoi est-il composé (Détailler via un exemple) ?
    Les ressources présentes sur le web sont localisées par un URL. Il s'agit d'une chaîne de caractères qui indique le protocole utilisé, le nom de domaine, et le chemin à suivre dans une arborescence pour accéder à la ressource sur le serveur.



    7. Qu’est-ce qu’un moteur de recherche ?
    Les moteurs de recherche sont des outils spécifiques du web destinés à fournir aux internautes des URL en adéquation avec les recherches qu’ils souhaitent effectuer.



    8. Expliquer l’usage et le fonctionnement des robots Crawler ou Spider.
    Ces moteurs de recherche utilisent pour cela des robots appelés crawlers ou spiders. Ils parcourent les ressources du web de lien hypertexte en lien hypertexte, afin de les analyser et d’en indexer le contenu.



    9. Expliquer comment un moteur de recherche propose des URL à partir de mots-clés de l’internaute.
    Un internaute qui souhaite accéder à une ressource web dont il ne connaît pas l’URL utilise un moteur de recherche en lui soumettant une requête sous la forme d’une série de mots-clés. En recoupant ces mots-clés avec sa base de données et à l’aide d’algorithmes, le moteur de recherche renvoie une liste d’URL compatibles, triées selon leur pertinence. Les moteurs de recherche ne sont pas tous équivalents : ils diffèrent selon les algorithmes qu’ils utilisent, le traitement des requêtes des internautes et la mise en avant de contenus publicitaires. Certaines entreprises choisissent ainsi un référencement payant.



    10. Qu’est-ce qu’une page web ? Que peut-elle contenir ? Nommer un langage pour créer une page web.
    Une page web est une ressource particulière du web pouvant accueillir du texte, des images, des vidéos, des sons et de nombreuses animations. Ces contenus sont mis en forme pour rendre la navigation agréable. Ces pages web sont codées à l’aide de différents langages.



    11. Comment appelle-t-on le système utilisé par le langage HTML ?
    Le langage HTML permet de créer et structurer le contenu de la page : texte, liens, images, titres, etc. Il s'agit d’un langage qui utilise un système de balises pour inclure tous ces éléments.



    12. Qu’est-ce que le CSS ?
    Le langage CSS permet de mettre en forme ces contenus. Il est utilisé dans des feuilles de style spécifiques, envoyées par les serveurs en complément du code HTML, pour définir les polices, les couleurs, les tailles, les décorations et certaines animations. D'autres langages sont aussi utilisés pour rendre les pages web dynamiques. La personnalisation peut utiliser des cookies, petits fichiers texte stockés par le navigateur de l’internaute, servant à retenir des identifiants, mots de passe, ou préférences de navigation.



    13. Est-ce que la réutilisation d’une page Web est toujours autorisée ? Nommer un modèle de licence.
    Le contenu des pages web est soumis à la législation des États. Même si leur publication ne nécessite pas d’autorisation préalable, les contenus peuvent être signalés s’ils enfreignent la loi. La propriété intellectuelle s’applique aussi. La réutilisation de certains contenus n’est pas toujours autorisée. Les conditions sont fixées par les auteurs, parfois sous des licences comme les Creative Commons.
